Fill out the Mad Lib






Addendum

This page was created as part of the ITC505 Lab 7 assignment. The goal was to implement a simple ExpressJS server and a Mad Lib form with at least five input fields. The form collects user input and generates a humorous story using the provided words.

Here is the step-by-step process:

  1. Setup ExpressJS Server: The `server.js` file initializes an Express server, sets up middleware for URL-encoded data and logging, and serves static files from the `public` directory. It also includes a route to generate a random number.
  2. Install Dependencies: We installed necessary packages: `express` for the server, `morgan` for improved logging, and `forever` for easier server management.
  3. Create Public Directory: A `public` folder was created to hold the HTML and CSS files for the labs, ensuring they are served correctly by the server.
  4. Create Mad Lib Form: The `index.html` file includes a form with five input fields: noun, verb, adjective, adverb, and plural noun. This form posts data back to the same URL.
  5. Handle Form Submission: In `server.js`, a POST route was added to handle form submissions. It takes the input from the form, constructs a Mad Lib story, and sends it back to the user.
  6. Style the Form: A `style.css` file was created to improve the appearance of the form and make the page more user-friendly.

By following these steps, the server and form were successfully set up, allowing users to create and view their Mad Lib stories dynamically.
